 Let's Get Technical

BACK END

I aim to develop applications that are scalable, reliable, and mantainable. I like to work with Node and the rich ecosystem of libraries that the NPM offers. I have built apps with the powerful Meteor framework, but I prefer classic Express. To use, or not to use a relational data model? No problem! I am comfortable with MongoDB and MySQL.

FRONT END

I strive to implement applications that look and feel great across devices. Anytime design is involved, I let my creativity run free to create elegant interfaces that feature a smooth user experience. I usually let Angular and jQuery do the power lifting, but I have a feeling I'll be trying out React and Vue in the very near future. Frameworks are sweet, but sometimes pure JS is the way to go.
